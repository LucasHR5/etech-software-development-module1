# Aula 1 – Introdução a Funções e Operações em Python

Nesta aula são apresentados conceitos iniciais da linguagem Python, focando no uso de funções, tipos de argumentos, formatação de saída e operações básicas.

## Funções em Python

Funções em Python são definidas utilizando a palavra-chave `def`. Elas permitem organizar o código, reutilizar lógica e tornar os programas mais legíveis.

São apresentados exemplos de funções simples que recebem parâmetros e retornam valores.

## Argumentos Posicionais

Argumentos posicionais são passados para a função seguindo a ordem definida nos parâmetros. A posição de cada valor é fundamental para o funcionamento correto da função.

Esse tipo de argumento é o mais comum e direto em chamadas de função.

## Argumentos Nomeados

Argumentos nomeados permitem informar explicitamente qual parâmetro está recebendo determinado valor. Com isso, a ordem dos argumentos deixa de ser obrigatória, aumentando a legibilidade do código.

## Operações Matemáticas em Funções

A aula apresenta o uso de operadores aritméticos dentro de funções, incluindo operações como soma e exponenciação, demonstrando como funções podem ser usadas para realizar cálculos.

## Uso da Função `print()`

É demonstrado o uso da função `print()` para exibir informações no terminal, incluindo o uso de parâmetros adicionais que permitem controlar o formato da saída, como separadores e quebras de linha.

## Formatação de Strings

São apresentadas diferentes formas de formatar textos em Python:
- F-strings, que permitem inserir variáveis diretamente em strings
- O método `format()`, incluindo o uso de parâmetros nomeados

Essas técnicas facilitam a exibição de mensagens dinâmicas e organizadas.

## Operações Aritméticas

A aula aborda os principais operadores aritméticos do Python, como soma, subtração, multiplicação, divisão, divisão inteira e módulo, aplicados a valores numéricos.

## Operações Relacionais

São introduzidos operadores relacionais, utilizados para comparar valores e retornar resultados booleanos (`True` ou `False`), como igualdade, diferença e comparações de maior ou menor.

## Operações Lógicas

Por fim, são apresentados operadores lógicos, que permitem combinar expressões booleanas utilizando `and`, `or` e `not`, possibilitando a criação de condições mais complexas.

## Objetivo da Aula

Apresentar os fundamentos iniciais de funções, argumentos, formatação de saída e operações em Python, servindo como base para os próximos conteúdos do curso.
