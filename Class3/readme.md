# Lista de Tarefas em Python

Este código implementa uma aplicação de lista de tarefas executada no terminal, utilizando Python.

Ele permite criar tarefas, armazená-las em memória, listar as tarefas existentes e alterar o status de uma tarefa por meio de um menu interativo.

---

## Funcionalidades

- Criação de tarefas com categoria, descrição, urgência e prazo
- Listagem das tarefas cadastradas
- Alteração do status das tarefas
- Remoção de tarefas concluídas
- Interação via terminal por meio de menu

---

## Organização do Código

- Uma classe representa os dados de uma tarefa
- Uma classe gerencia a coleção de tarefas
- Um loop principal controla a execução do programa e o menu de opções

---

## Execução

O programa permanece em execução até que o usuário escolha a opção de sair.
